# Execute on dev

1. Clone the repo
2. Execute 
```
npm i or npm install
```
3. You have to need Nest CLI installed
```
npm i -g @nest/cli
```
4. Up database
```
docker-compose up -d
```


## Stack used
* Mongodb
* Nest
